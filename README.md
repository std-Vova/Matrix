# Matrix_
Репозиторий содержит тесты и шаблонный класс, реализующий математическую матрицу. Класс предоставляет как основные алгоритмы и опрерации над матрицами, так и продвинутые способы сложения, умножения матриц, которые выполняются в несколько раз быстрее стандартных. Достигается это при помощи использования мета - програмирования, fold - expression, CTRP, expression templates. Так же экземпляры могут содержать в себе не только стандартные типы, но и длинную арифметику big_integer and rational.
